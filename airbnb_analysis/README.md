# Airbnb Market Analysis: Columbus vs New York

## Author
Annie Zahoran 

## Project Overview
This project analyzes Airbnb listing data to compare the Columbus, Ohio and New York City markets. The goal is to help a traveler make the best decision on where to stay affordably and comfortably.

## Research Questions

1. What neighborhood has the best value in New York and Columbus?
2. What neighborhood has the highest customer satisfaction? 
3. Which room types on a listing drive higher prices?
4. How do prices and availability vary across neighborhoods within cities? 
5. Do recent reviews and amount of reviews create more bookings?
